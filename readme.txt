1 - Necessario instalar o node.js e o visual studio code
2 - Depois de instalados é necessario abrir o projeto no visual studio na pasta raiz
3 - Então abrir o terminal e instalar o cypress "npm install --save-dev cypress@9.7.0"
4 - Instalar o plugin para utilizar o cucumber "npm install -D cypress-cucumber-preprocessor"
5 - Abrir o cypress para exucutar os testes com o comando "npx cypress open"
6 - Entao selecionar os testes para a execução
